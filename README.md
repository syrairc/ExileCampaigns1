# ExileCampaigns

A leveling companion overlay. Follow a step-by-step campaign route without alt-tabbing to a guide.

## What it does

- **Route guide** - ordered steps per act, with on-screen pathing and minimap markers pointing you at the next objective.
- **Auto-advance** - steps tick off as you enter areas and finish quests; a backtrack hold lets you review earlier steps without it snapping forward.
- **Waypoint helper** - on a travel step, highlights which waypoint node to click on the world map.
- **Build planner** - import a build from a link or pasted code, then highlight planned items in vendor, stash, and reward windows with per-item tooltips.
- **Stats** - live XP and leveling-penalty readout.
